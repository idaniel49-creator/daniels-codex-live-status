# Daniel's Codex Live Status

Public read-only status feed for Daniel's Codex Control App.

This repository intentionally contains only sanitized status data that can be shown in Daniel's iPhone/Lovable dashboard.

It must not contain:

- Secrets, tokens, cookies, auth files, or account files
- Private memory/context files
- Local save files or browser profiles
- Game assets, binaries, caches, or large files
- Personal notes that are not meant to be public

Main feed:

```text
status.json
```

Raw URL:

```text
https://raw.githubusercontent.com/idaniel49-creator/daniels-codex-live-status/main/status.json
```
